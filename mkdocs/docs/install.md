# Install

## Prérequis

Il faut avoir Java 11 d'installé sur votre machine.

## Linux and Mac

- Télécharger l'archive Zip
- Extraire le contenu de l'archive
- Se positionner dans le dossier bindist/bin
- Executer le fichier run

## Windows

- Télécharger l'archive Zip
- Extraire le contenu de l'archive
- Se positionner dans le dossier bindist/bin
- Executer le fichier run.bat
