# Usage

## Lancement du programme

### Fournir les données

Afin de pouvoir apporter les informations nécessaires à la simulation, vous pouvez ajouter le volume total de la baignoire ainsi que le débit du robinet
dans les champs destinés.

Une fois ceci fait vous pouvez choisir le nombre de trous à créer, une fois la valeur choisie, de nouveaux champs apparaîtrons avec les débits des trous à remplir.

Si vous saissisez des valeurs non correctes, l'application vous recommandera de resaisir vos valeurs.

### Lancement de la simulation

Si vous souhaitez lancer la simulation une fois que vous avez remplis les champs il suffit d'appuyer sur le bouton "lancer la simulation".

### Stopper l'application

Si vous souhaitez arrêter l'application, il suffit d'appuyer sur "Arrêter la simulation" et ainsi les informations finales de la simulation apparaîtront tout en bas de l'application.
